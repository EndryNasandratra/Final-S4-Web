SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

Description

Contrôle la maniere dont le document sera affiche par le lecteur. On peut regler le niveau de zoom : afficher la page en entier, occuper toute la largeur de la fenetre, utiliser la taille reelle, choisir un facteur de zoom particulier ou encore utiliser la valeur par defaut de l'utilisateur (celle parametree dans le menu Preferences d'Adobe Reader). On peut egalement specifier la disposition des pages : une seule a la fois, affichage continu, pages sur deux colonnes ou reglage par defaut.

Parametres

zoom
Le zoom a utiliser. Il peut prendre l'une des valeurs chaînes suivantes :
  • fullpage : affiche entierement les pages a l'ecran
  • fullwidth : maximise la largeur des pages
  • real : affiche la taille reelle (equivaut a un zoom de 100%)
  • default : utilise le mode d'affichage par defaut du lecteur
ou bien un entier indiquant le facteur de zoom a utiliser.
layout
La disposition des pages. Les valeurs possibles sont :
  • single : affiche une seule page a la fois
  • continuous : affichage continu d'une page a l'autre
  • two : affiche deux pages sur deux colonnes
  • default : utilise le mode d'affichage par defaut du lecteur
La valeur par defaut est default.
