Header

Header()

Description

Cette methode permet de definir l'en-tete de page. Elle est appelee automatiquement par AddPage() et ne devrait donc pas etre appelee explicitement par l'application. L'implementation de Header() dans FPDF est vide, donc vous devez deriver la classe et redefinir la methode si vous voulez un traitement particulier pour vos en-tetes.

Exemple

class PDF extends FPDF
{
    function Header()
    {
        // Police Arial gras 15
        $this->SetFont('Arial', 'B', 15);
        // Decalage
        $this->Cell(80);
        // Titre encadre
        $this->Cell(30, 10, 'Titre', 1, 0, 'C');
        // Saut de ligne
        $this->Ln(20);
    }
}
